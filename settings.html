🔍 Detail Auditor (sister)

A second pass, once per batch: it checks whether the compact snippet dropped important specifics (exact numbers, named plans/tactics, promises, capabilities, identities) and, if so, attaches a short detail note to that snippet — injected alongside the summary and never auto-edited. Empty for routine batches. Uses the same connection as the summarizer. View / edit / regenerate / delete each note in the Snippet Browser.

🔗 Copilot Bridge (Author's Note)

External OOC tools like ST-Copilot can't read Summaryception's memory directly — they only read native fields. These buttons copy your full memory (notepad + every snippet + details) into the Author's Note so Copilot can see the ghosted past too, then let you pull it back out. Your existing Author's Note is stashed in a non-injected slot and returns on Restore. Note: while mirrored, the memory is briefly also sent to your roleplay AI (harmless — it's your own canon), so Restore when you're done. Tip: if you set your Author's Note to not inject (disable it), test whether Copilot still sees it — if so, you get zero duplication.
Requires Debug Mode to be enabled.
